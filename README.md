# Pho2Vis
使用深度学习模型，基于全天空成像仪所拍摄的圆形天空图片进行大气能见度预测

## 目前进度
1. 实现了读取数据集的Dataset类
2. 微调预训练模型DenseNet121
3. 整体训练和评估模型的大概框架已完成
